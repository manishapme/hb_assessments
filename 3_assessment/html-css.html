Skills: HTML and CSS

Part One: HTML

Add to the HTML document application-form.html so that it has the following items.

  1. A form that asks the user:
    • Their first and last name (as separate fields),
    • Their salary requirement as a number, and
    • Which job they are applying for, from a list of:
      • Software Engineer
      • QA Engineer
      • Product Manager
  2. The form should also have a submit button that submits the form to the URL path /application. This should be a POST request. (For now, just focus on the HTML requirements for this request. Don’t build a Flask route, yet.)
  3. The page should have a large heading and some explanatory text, as well.
  4. Add the elements to make your HTML form accessible to screen readers. For more information, see http://webaim.org/techniques/forms/controls.

Part Two: CSS

Create a CSS file, styles.css, that stylizes application-form.html.

First, edit the HTML file to reference the CSS file so it’s used to stylize the HTML.

Complete each of the following styles using CSS:

  • Change the background color of all of the form input boxes and submit button to be yellow (can you do this for all input boxes without having to stylize each one independently?)
  • Change the text in the last name field to be bold.
  • Draw a border around the instruction box.
